Technical Documentation for Clean Installation of Ubuntu, CUDA, cuDNN, and PyTorch on WSL2

This guide provides step-by-step instructions for setting up a clean installation of Ubuntu on Windows Subsystem for Linux 2 (WSL2), followed by installing CUDA, cuDNN, and PyTorch for machine learning development.

Prerequisites

  • Windows 10 version 2004+ or Windows 11.
  • An NVIDIA GPU compatible with CUDA 12.4.
  • Administrative access and at least 20 GB of free disk space.

Core Setup Steps

  1. Remove unused WSL distributions and set WSL2 as default.
  2. Install Ubuntu 24.04 and basic development tools.
  3. Install CUDA 12.4, then configure CUDA_HOME, PATH, and LD_LIBRARY_PATH.
  4. Install cuDNN and verify compatibility with the CUDA version.
  5. Create a Python virtual environment and install PyTorch with CUDA support.
wsl --set-default-version 2
wsl --install -d Ubuntu-24.04
sudo apt update && sudo apt upgrade -y
sudo apt install -y build-essential git wget curl
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124
